Packages

R ist in Packages organisiert. Wichtig ist die Installation der ersten Packages - wir werden dieses Thema direkt im nächsten Schritt aufgreifen und die Pakete häufig verwenden! Dabei verwenden wir die Funktion für die Verwaltung der Packages in RStudio. Wir installieren im ersten Schritt die folgenden Packages:

  • RColorBrewer
  • MASS (könnte bereits installiert sein, testen Sie dies mit library(MASS)
  • tidyverse
  • ISLR aus einem lokalen zip-Archiv (Links finden Sie unten!)

Packages können wir auch aus einem lokalen zip- oder tar-Archiv installieren. Als Beispiel installieren wir das ISLR-Paket. Sie erhalten dieses Package inzwischen auch auf CRAN, aber wir üben an Hand dieses Beispiels die Installation aus einer zip-Datei. Wählen Sie das passende Download für ein Binary auf der Seite im folgenden Link:

ISLR-Paket

Das Package Tidyverse ist ein sehr mächtiges und performantes Werkzeug. Insbesondere ggplot2 für die grafische Ausgabe und dplyr für die Manipulation von Daten werden wir häufig einsetzen. In dieser Menge von Packages finden wir veraussichtlich die Zukunft für R. Am Besten installieren Sie das komplette Package.
