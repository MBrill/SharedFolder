SciPy

SciPy ist eine Sammlung von Modulen, die auf NumPy aufbaut und mathematische Aufgaben löst - von der Lösung von linearen Gleichungssystem über die Näherung für Nullstellen von Funktionen bis hin zur numerischen Lösung von Differentialgleichungen.

SciPy

SymPy

Neben numerischen Berechnungen kann man auf dem Computer auch "symbolisch" rechnen. Die Software "Mathematica", aber auch "Maple", sind sehr bekannte Vertreter solcher symbolischen Berechnungspakete. Eine andere Bezeichnung für eine Software, mit der wir symbolisch rechnen können ist Computeralgebra-System.

In Python können wir dies auch - mit Hilfe des Moduls SymPy.
